设计说明

01 · 语义

A 即 Athlon

字母 A 是品牌首字母；横杠处的终端提示符 > 暗示 Agent 与命令行，一行两义。

02 · 克制

单笔单渐变

无投影、无星花、无人脸。仅一道渐变贯穿字身与提示符，保证 16px 下仍可辨。

03 · 落地

SVG 单源

以 <symbol> 定义一次，<use> 复用于各尺寸；可直接导出为 app-icon.svg。

Athlon Agent · 简洁图标方案 · 与 现有 app-icon.svg 相比去除了机器人脸与星花装饰。
